# Annapurna Home Foods - Frontend

E-commerce storefront built with React, Vite, TypeScript, Tailwind CSS, and shadcn-ui.

## Local Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

## Deploy to GitHub Pages

```sh
npm run deploy
```

Deployed URL:

`https://vinaykalacharla.github.io/annapurna-home-kitchen-ui/`
